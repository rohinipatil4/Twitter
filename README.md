# To use jacoco for code coverage, 

1. add following plugin in build.gradle

    apply plugin: "jacoco"

2. run below commands 

   ./gradlew clean
   ./gradlew build

3. code coverage report will be seen on 

   file://<user-directory>build/reports/tests/test/index.html
